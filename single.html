World Happiness Data Analysis

Analysis of World Happiness Scores reported in the 2020 World Happiness Report using Python, R and Tableau.

April 2021 Diondra Stubbs

This project analyzes the 2020 World Happiness Report to draw conclusions about the general well being of 153 countries in the world. The analysis is performed in Python.

The main dataset used includes over 150 countries and their happiness scores, freedom to make life choices, social support, healthy life expectancy, regional indicator, perceptions of corruption and generosity.

The project includes some data wrangling and exploratory data analysis in Python. It also involves a regression analysis in Python and visualizations in Tableau.
